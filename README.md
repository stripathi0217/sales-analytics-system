### Sales Analytics System

This repository contains a complete Sales Analytics System implemented in Python as part of an academic assignment.
The system processes raw sales data, performs validation, analysis, API enrichment, report generation, and executes as a full application.

### Setup Instructions

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. How to Run the Complete System
   python -m Q6.main

### Outputs Generated

1. Enriched sales data:
   path - Q6/data/enriched_sales_data.txt
2. Final sales report:
   path - Q6/output/sales_report.txt
