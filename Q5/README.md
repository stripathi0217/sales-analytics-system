### Q5 – Report Generation

This folder contains the solution for Question 5: Report Generation of the Sales Analytics System assignment.
The goal is to generate a comprehensive, well-formatted text report summarizing all analytics and insights from the sales data.

- Open Visual Studio Code
- Open the Q5 folder
- Open the terminal
